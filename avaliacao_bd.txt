Avaliação de bancos de dados:

Você está criando a base de dados para um novo sistema de blog. A estrutura inicial deve permitir que autores publiquem conteúdo e que esse conteúdo seja organizado por tags ou categorias. Modele a estrutura de um blog simples, onde usuários podem escrever posts e categorizá-los.

Regras de Negócio:

1 - O sistema deve armazenar Usuários (que são os autores dos posts). Deve-se guardar (ID, Nome, Email, Senha).

2 - O sistema deve armazenar Posts (os artigos). Deve-se guardar (ID, Título, Conteúdo (texto longo), Data de Publicação).

3 - O sistema deve armazenar Categorias (para organizar os posts). Deve-se guardar (ID, Nome da Categoria).

4 - Um Usuário pode escrever vários Posts.

5 - Cada Post é escrito por exatamente um Usuário.

6 - Um Post pode pertencer a uma ou mais Categorias.

7 - Uma Categoria pode estar associada a vários Posts.

Instruções da Avaliação:

Modelo Conceitual (Draw.io):

Crie um Diagrama de Entidade-Relacionamento (DER) no Draw.io atentando-se para o uso correto das formas de entidades, atributos, relacionamentos e notações.

Identifique todas as entidades, seus principais atributos e as chaves primárias (PK).

Desenhe os relacionamentos entre as entidades, especificando a cardinalidade (1:1, 1:N, N:M).

Resolva o relacionamento N:M com uma entidade associativa.

Exporte o diagrama como uma imagem (PNG ou JPG) ou PDF.

Modelo Lógico (MySQL Workbench):

Abra o MySQL Workbench e crie um novo Modelo EER (diagrama).

Traduza o seu Modelo Conceitual para um Modelo Lógico.

Crie as tabelas correspondentes às suas entidades e entidades associativas.

Defina todos os atributos (colunas) com os tipos de dados MySQL apropriados (ex: INT, VARCHAR(255), TEXT, DATETIME). **Muita atenção nesta etapa!**

Configure corretamente todas as Chaves Primárias (PK) e Chaves Estrangeiras (FK) para implementar os relacionamentos.

Salve o modelo (arquivo .mwb, não será aceito outro tipo de arquivo para o modelo lógico!).

Você deve entregar 2 arquivos:

O arquivo de imagem (ou PDF) do Modelo Conceitual (Draw.io).

O arquivo .mwb do Modelo Lógico (MySQL Workbench).
